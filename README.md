# AIE-BAN
The code of the paper " Adaptive Image Enhancement based on Brightness-aware Network for Non-uniform Lighting Image"
## Requirements
1. Python 3.7 
2. Pytorch 1.0.0
3. opencv
4. torchvision 0.2.1
5. cuda 10.0

### Test: 

```
python lowlight_test.py 
```

### Train:
```
python lowlight_train.py 
```
